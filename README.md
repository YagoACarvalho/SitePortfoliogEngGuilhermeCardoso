# 🏗️ Portfólio Eng. Guilherme Cardoso

Portfólio moderno desenvolvido em **React + Vite + TailwindCSS**, com foco em apresentação profissional de serviços de engenharia civil.

O projeto foi pensado para sair do padrão tradicional e trazer uma abordagem mais atual, visual e orientada a conversão.

---

## 🚀 Tecnologias utilizadas

* ⚛️ React
* ⚡ Vite
* 🎨 TailwindCSS
* 🎬 Framer Motion (animações)
* 🎯 Lucide React (ícones)
* 🔥 React Icons (ícones adicionais)

---

## 📸 Preview

Interface moderna com:

* Hero section impactante
* Apresentação de serviços
* Estrutura de projetos (cases)
* Seção de processo (passo a passo)
* Call to Action com WhatsApp

---

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/portfolio-eng-guilherme.git
```

Acesse a pasta:

```bash
cd portfolio-eng-guilherme
```

Instale as dependências:

```bash
npm install
```

---

## ▶️ Rodando o projeto

```bash
npm run dev
```

Acesse no navegador:

```
http://localhost:5173
```

---

## ⚙️ Configuração

### 📲 WhatsApp

Altere o link no arquivo:

```
src/App.jsx
```

Substitua:

```js
https://wa.me/5500000000000
```

Por:

```js
https://wa.me/55SEUNUMERO
```

---

### 🖼️ Imagens

Atualmente o projeto utiliza imagens do Unsplash.

Recomendado:

* Substituir por fotos reais das obras
* Ou adicionar imagens em:

```
/public
```

Exemplo:

```jsx
<img src="/obra1.jpg" />
```
